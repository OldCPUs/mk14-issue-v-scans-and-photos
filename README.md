# MK14 Issue V Scans and Photos

This repository preserves high-quality scans and photographs of the Sinclair/Science of Cambridge MK14 Issue V microcomputer.  
It is intended as a visual reference for researchers, hardware enthusiasts, restorers, and emulator developers.

## Contents

- Flatbed scans of the MK14 Issue V printed circuit board (front and back)

## Purpose

To contribute to the long-term preservation and accessibility of MK14 hardware details, with an emphasis on:

- Historical accuracy and visual clarity
- Reference support for PCB recreation or emulation
- Community access to hard-to-find visual material
